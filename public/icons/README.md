# AI 劳动记录器 - 图标占位文件

这些是占位图标文件，在发布扩展之前需要替换为实际的 PNG 图标。

## 所需尺寸

- icon16.png（16x16 像素）
- icon32.png（32x32 像素）
- icon48.png（48x48 像素）
- icon128.png（128x128 像素）

## 建议

建议使用 Figma、Photoshop 或在线图标生成工具创建符合 AI 劳动记录器品牌风格的统一图标。

## 图标设计指南

- 保持简洁明了的设计风格
- 使用与扩展主题一致的配色方案
- 确保在小尺寸下也能清晰辨识
- 建议使用 SVG 源文件导出各尺寸 PNG
